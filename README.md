# 旋转体STL模型生成器

一个功能强大的网页工具，用于生成旋转体STL 3D模型。支持交互式2D坐标系绘制和实时3D预览。

## 功能特点

- 🎨 **交互式2D坐标系**：点击坐标系添加轮廓点，可视化编辑
- 🔄 **360度旋转体生成**：将2D轮廓绕y轴旋转360度生成3D模型
- 👁️ **实时3D预览**：生成后立即预览模型，支持拖动、旋转、缩放
- 📝 **文本输入**：支持手动输入坐标点
- 💾 **STL文件下载**：一键下载生成的STL文件
- 🎯 **自动法线修正**：确保所有面的法线方向正确，避免渲染问题

## 使用方法

### 🌐 在线使用（推荐）

如果已发布到GitHub Pages，可以直接访问在线版本：
- 访问：`https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/` 或
- 访问：`https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/index.html`

### 💻 本地使用

1. 下载或克隆此仓库
2. 打开 `index.html` 或 `stl_generator_with_preview.html` 文件
3. 在2D坐标系上点击添加轮廓点，或在文本框中输入坐标
4. 设置旋转分段数（推荐64-256）
5. 点击"生成并下载STL"生成文件
6. 点击"预览模型"查看3D预览

### 坐标输入格式

每行一个点，格式为 `x, y` 或 `x y`：
```
0, 0
-35, 0
-35, 5
-25, 5
-25, 20
-20, 20
-20, 10
0, 10
```

- **x坐标**：水平方向（旋转轴为y轴）
- **y坐标**：垂直方向（旋转体的高度）

### 交互式坐标系操作

- **左键点击**：添加点
- **右键点击**：删除最后一点
- **双击**：清空所有点
- **从文本框同步**：点击"从文本框同步"按钮

### 3D预览控制

- **左键拖动**：旋转模型
- **右键拖动**：平移模型
- **滚轮**：缩放模型
- **双击**：重置视角

## 文件说明

- `index.html` - **主文件**（完整版，带3D预览和交互式坐标系）- 推荐使用
- `stl_generator_with_preview.html` - 完整版（带3D预览和交互式坐标系）
- `stl_generator.html` - 简化版（仅生成STL文件）
- `generate_ring.py` - Python脚本：生成圆环STL模型
- `generate_tray.py` - Python脚本：生成托盘旋转体STL模型
- `generate_tray_half.py` - Python脚本：使用轮廓一半生成旋转体

## 技术栈

- **Three.js** - 3D渲染和预览
- **HTML5 Canvas** - 2D坐标系绘制
- **纯JavaScript** - 无需服务器，可直接在浏览器运行

## 浏览器兼容性

- Chrome/Edge（推荐）
- Firefox
- Safari
- 需要支持WebGL的现代浏览器

## 注意事项

- 建议从旋转轴（x=0）开始绘制轮廓，形成闭合轮廓
- 旋转分段数越大，模型越平滑，但文件也越大
- 如果CDN加载失败，请检查网络连接或使用离线版本
